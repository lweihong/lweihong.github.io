JavaScript运算符

  • 算数运算符
    • + - * / %
  • 一元运算符
    • 一元指的是操作数的个数 ++ --
      • 前置和后置不同:
        • 前置:先自增（减） 然后参与当前运算
        • 先参与运算，再自增(减)

  • 逻辑运算符

    • 操作数都是bool值时

      • 逻辑与(且) && 两个操作数都为true结果为true
      • 逻辑或|| 有一个为ture就为true，两个都为false，结果为false
      • 取反 ！

      • 操作数不是bool值时： 短路操作

  • 比较运算符

    • > < >= <= != !== 返回结果为bool类型
  • 赋值运算符
    • = += -= *= /= %=
  • 相等运算符
    • == 值相等 === 值和类型全等
  • 运算符的优先级
    • 从高到低：()优先级最高 > 一元运算符（++ --）>算数运算符（先*/%后+-）-> 关系运算符（> >= < <=）> 相等运算符（== ！= === ！==）>逻辑运算符（先 && 后||）
