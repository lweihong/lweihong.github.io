DOM

JavaScript基础篇
  • document.getElementById()
  • document.getElementsByTagName()
  • DOM(document object model)文档对象模型
  • 页面元素: 页面中的标签

  • 根据id获取页面元素:

    • document.getElementById(“box”) get获取 element 元素 by通过 id 标签的唯一标识,获取的id名为字符串格式！
    • 由于html是从上往下执行的，如果将script中获取标签的语句写在head标签中，导致获取不到，我们可以通过window.onload这种形式去更改页面加载顺序,window.onload中的代码会在页面完全加载后执行。

  • 更改标签样式：

    • 第一步获取标签，可以保存在变量中
    • 第二步通过标签的style属性去访问样式中的某个具体样式
    • 设置的样式值是一个字符串形式。设置样式时，style不要忘记！通过js设置的样式是设置在行内的。
    • 设置标签的文本内容
      标签.innerHTML = “hello,world!”;

  • 根据标签名获取页面元素

    • document.getElementsByTagName(“div”);
    • 由于获取结果可能是多个，element后面要加s,根据标签获取的结果是伪数组形式，伪数组是不具备数组的方法。要操作伪数组中的所有元素需要遍历伪数组。
    • 根据标签名获取元素时，有可能获取到的标签只有一个，但是形式还是伪数组。
    • 根据标签名获取时，document可以更换为任意标签。
    • 如果标签具有了某些属性，我们可以直接通过.属性名的形式访问属性值
    • body的访问方式 document.body.style.backgroundImage = “url()”;

  • 样式相关

    • 访问标签的类名属性通过 标签.className。

    • 通过 标签.style.width 只能获取行内设置的样式属性，内嵌式和外链式都无法获取。getComputedStyle(标签,false).width 可以获取计算后的样式
      ie低版本不支持getComputedStyle，IE获取行内样式属性：标签.currentStyle.height，可以封闭一个函数兼容两种获取方式：

    • 文本的设置：
      通过box.innerText可以获取不含内部标签的文本，ie支持;
      其他浏览器支持box.textContent属性，作用与innerText相同;
      box.innerHTML 可以获取和设置某个标签内的文本和内部标签;

* 标签的获取方式
document.getElementById(“id名”) 根据id获取
document.getElementsByTagName(“标签名”)   根据标签名获取
document.getElementsByClassName(“类名”) 根据类名获取  ie低版本不支持
* 封装根据类名获取页面元素的函数
