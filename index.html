数组

  • 数组可以用来存储多个数据
  • 数组的声明方式：
  • 数组中的每一个数据我们称为元素;
    • 由于js是弱类型，一个可以存储不同类型的元素.
    • 一般不推荐在数组中保存多个类型的数据.
  • 数组的长度属性: arr.length数组中元素的个数;

  • 数组的索引

    • 数组中的元素是按照索引排列的，索引从左往右计算，第一位为0;

  • 数组遍历

    • 向数组最后添加一个新元素索引值为arr.length arr[arr.length];
    • 数组的长度是随着数组元素的增减而增减的。
    • 数组的长度可以进行设置，如果设置的长度大于数组元素个数，不影响内容，多出的部分值为 undefined,如果设置的长度小于了元素实际个数，超出的部分元素会被删除。

  • 数组的类型

    • 数组是一个复杂数据类型，通过typeof检测时得到object，不准确，我们可以通过下列方法去检测数组的类型。
  • 基本数据类型和复杂数据类型的区别
    • 直观区别，数据量不同。
    • 保存上的区别：
      • 基本数据类型的值保存在栈中
      • 复杂数据类型的值保存在堆中
    • 复制的区别：
      • 基本数据类型复制的是栈中的值，在内存中开辟新区域，两者是完全独立的；
      • 复杂数据类型复制的是保存在栈中的指针，两个变量指向的是堆中的同一个区域；

JavaScript变量

  • 变量的作用：存储数据
  • 数据分为两种：
    • 固定数据：固定的数据保存在硬盘中的。
    • 临时数据：临时的数据是保存在内存中。
    • 数据是保存在内存中的某一个区域的，每个区域对应一个内存地址，我们想要使用某个区域中的数据，就需要找到指定的地址，使用起来不方便。我们可以通过变量，变量实际上保存的是某个数据的地址，我们使用变量实际上就是使用变量中保存的地址中的数据。
  • 变量的使用
    • 变量的声明：
      • var 变量名，var—variable可变的 js是弱类型语言，变量的类型不定，可变
    • 变量的命名规则：
      • 1、不能使用关键字和保留字
      • 2、开头可以使用_$英文字母
      • 3、除了开头以外可以使用数字
      • 4、不能包含空格
      • 5、严格区分大小写;
    • 变量的赋值和初始化
      • 初始化：在变量声明同时进行赋值 var num = 100;
      • 赋值： var num = 100; num = 10; 由于js是弱类型，除了保存数值型的变量也可以保存其他类型数据;但是我们不推荐这样做，可能会对使用造成不良影响。

  • 变量的类型

    • 基本数据类型：字符串(string); 数值型(number); 布尔类型(boolean); undefined; null.

      • 字符串的值：使用成对的双引号或者单引号进行包裹(js中所有的标点符号必须为英文半角);
      • 数值型的值：例如:0-9,0.1 0.2
        • NaN (not a number) 不是一个数值类型
        • isNaN(数据); 是不是不是数 如果判断的数据不是数值型返回true
        • Infinity 表示无穷大，超出了js能承受的最大值
        • -Infinity 表示无穷小
      • 布尔类型
        • true 真
        • false 假
      • undefined
        • defined 定义
        • undefined 出现的情况： 变量声明未赋值

      • null 空

        • undefined 与 null 相等不全等

      • 数据类型判断

        • typeof 变量 / typeof(变量)

  • 变量的类型转换

    • 将其他类型转换为字符串类型：

      • 强制转换

      • 隐式转换

        • 通过任意类型+任意字符串操作可以将其他类型转换为字符串类型
        • 在使用+操作的时候为了不改变数据的原有内容，使用空字符串
        • 空字符串中不能有任何内容
      • 基本类型值的不可变性
        • 通过类型转换或者重新赋值都无法修改基本类型的原有值。

    • 将其他类型转换为数值型：

      • 强制转换
        • Number() 只能转换全部是数值型的字符串
        • parseInt() 可以转换前半部分为数字的字符串，转换结果为整数
        • parseFloat()可以包含小数
      • 隐式转换
        • */ % 可以进行转换
        • +操作需要将+号放在数据前进行使用。
    • 将其他类型转换为布尔类型：
      • 强制转换
        • Boolean()
          • 0 undefined null “” NaN 转换值为false
          • 有值的字符串 有值的数 转换为true
      • 隐式转换
        • ！！ 可以进行隐式转换
    • undefined和null类型只有一个值，不需要转换。
